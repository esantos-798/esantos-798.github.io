Por que IA + OTN?

Cascata a partir de um único dLOS
— fixo (independe de N) —

1× ODU-AIS + 1× ODU-BDI
1×6× TCMi-AIS + 1×6× TCMi-BDI
1× Client-AIS
Total: 19 defeitos
N ≥ 2 (+ agregador ODU)
(N+1)× ODU-AIS + (N+1)× ODU-BDI

N× Client-AIS
N=2 → 48  ·  N=4 → 78 defeitos
O desafio
Redes DWDM modernas operam centenas de λ simultâneos. Um único dLOS gera cascata de defeitos em múltiplas camadas em milissegundos — gestão manual é inviável com SLA de 99,999%.
Este artigo
Um dos primeiros trabalhos a aplicar deep learning especificamente para predição de alarmes em redes OTN — comparando três arquiteturas com dados reais e balanceamento SMOTE.

Três arquiteturas em competição.

LSTM
Long Short-Term Memory
Rede recorrente com memória seletiva unidirecional. Gates de entrada, esquecimento e saída controlam o que é retido ao longo da sequência. Aprende: "depois deste padrão de defeitos, o próximo tende a ser X".
~88%acurácia
F1 ~0,82(estimado)
BiLSTM
Bidirectional LSTM
Processa a sequência nos dois sentidos simultaneamente: uma LSTM no sentido passado→futuro e outra no sentido futuro→passado. As duas saídas são concatenadas, capturando contexto bidirecional de cada evento.
~91%acurácia
F1 ~0,87(estimado)
LSTM-XGB
LSTM-XGBoost Híbrido ★ melhor
LSTM extrai representações temporais das sequências; essas features são passadas ao XGBoost para classificação final com gradient boosting. Combina o poder temporal da LSTM com a robustez do XGBoost.
94,24%acurácia
F1 = 0,91

// 04 — impacto do SMOTE OTN + LSTM · IEEE ICWT 2025 11 / 14

SMOTE — por que faz diferença.

O que é SMOTE

Synthetic Minority Over-sampling Technique gera amostras sintéticas da classe minoritária por interpolação linear entre exemplos reais existentes. Para cada amostra de defeito real, cria vizinhos artificiais no espaço de features.

Por que é necessário

Sem balanceamento, modelos de ML são enviesados para a classe majoritária. Em redes OTN bem operadas, defeitos críticos são raros — exatamente os eventos mais importantes de detectar. SMOTE força o modelo a aprender essas fronteiras de decisão.
